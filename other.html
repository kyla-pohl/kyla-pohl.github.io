Internet Recommendations

  • The MathSciNet collaboration distance tool describes the authorship relationship between any two published mathematicians. In particular, this is great for finding Erdős numbers.
  • Detexify is an excellent tool for LaTeX learners. How it works: draw symbols that you don't know how to TeX. Detexify will guess what it thinks you've drawn and then provide code for doing it in your document.
  • In a similar vein, TeXnique is a game which tests your LaTeX skills.
